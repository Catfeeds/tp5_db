公司简介Company Profile

公司总经理蛇佬大12岁师从其父亲开始学习蛇医技术，孜孜不倦刻苦钻研5年后在充分掌握传统医学的基础上并有多项摸索， 创新蛇医绝技冠绝方园百里，多年来利用其所学救死扶伤，以为百余名毒蛇咬伤患者和无数的皮肤病患者布药施治，让 伤者摆脱死亡的威胁和病痛的折磨而重新投入生活工作。为蛇文化的赓续传承发展奠定了扎实的基础，并为发扬光大， 蛇佬大继承父亲蛇医绝技衣钵的基础下，于2014年8月成立了杭州蛇佬大生态蛇基地，（这也就是公司的前身） 基地从事五步蛇、蝮蛇、乌梢蛇、王锦蛇、银环蛇等蛇类的驯养繁殖、加工及蛇产品的深度开发展示,蛇类技术培训科普教育， 以蛇为主题的休闲旅游、文化娱乐... ...
